# 印刷品设计制作指南

## 品牌视觉规范

### 色彩
| 用途 | 色值 | Tailwind |
|------|------|----------|
| 主色（强调/线条/图标） | `#DC2626` | red-600 |
| 强调色（数据高亮） | `#F59E0B` | amber-500 |
| 深色背景 | `#0F172A` | slate-900 |
| 卡片/区块底色 | `#1E293B` | slate-800 |
| 正文文字 | `#0F172A` | slate-900 |
| 浅色背景 | `#F5F5F5` | — |
| 白色表面 | `#FFFFFF` | — |

### 字体
- 英文标题: Inter Extra Bold (800)
- 英文正文: Inter Regular (400) / Medium (500)
- 关键数字: Inter Extra Bold (800) — 大号、红色或琥珀色
- 最小正文字号: 9pt（印刷）/ 8pt（注释/免责声明）
- 中文: 思源黑体 Noto Sans SC

### Logo 使用
- 印刷需要矢量格式 (AI/EPS/SVG)
- 当前仅有 696×679 PNG — 不可用于印刷
- 最小印刷尺寸: 宽度 ≥ 25mm
- 深色底用反白版本（如有），浅色底用标准版本

### 间距
- 页面边距: 10-15mm
- 段落间距: 正文行距的 1.5-2 倍
- 安全区（出血线内）: 5mm

## 内容撰写原则

1. **标题用数字** — "3x Faster" 好于 "Very Fast"
2. **每段 ≤ 3 句话** — 工业买家扫描阅读，不会逐字读
3. **一个跨页 = 一个核心信息** — 不要在一个版面上塞多个主题
4. **对比数据用表格** — 混气 vs O2/Air 的对比放在表格里最直观
5. **每页都要有 CTA** — 网站、邮箱、二维码至少出现一个
6. **免责声明用小字** — ROI 数据、切割速度等标注测算条件

## 制作流程

```
内容定稿 → 设计 → 校对 → 打样确认 → 批量印刷
  1周       2周     1周      3天        1-2周
```

### 打样检查清单
- [ ] 色彩还原（红色是否偏橙？深色底是否发灰？）
- [ ] 文字清晰度（小字是否有锯齿？）
- [ ] 图片清晰度（300dpi 图片是否被拉伸？）
- [ ] 裁切/出血（边距是否被切到？）
- [ ] 装订（骑马钉位置是否偏移？）
- [ ] logo 清晰度（PNG 拉大后是否模糊？）
