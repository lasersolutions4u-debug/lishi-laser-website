# B2B 工业平台运营

> 状态: 待启动
> 目标平台: ThomasNet (北美), DirectIndustry (欧洲)
> 配合: LinkedIn (P0, 主力渠道) — B2B 平台为被动获客补充，LinkedIn 为主动触达主力

## ThomasNet (美国)

- 网址: thomasnet.com
- 定位: 北美最大工业采购目录
- 用户: 美国制造业采购经理和工程师
- 模式: 供应商注册 → 产品上架 → 买家搜索发现 → 询盘

### 注册准备
- [ ] 公司信息（英文）
- [ ] 产品分类: Laser Cutting Equipment / Gas Systems
- [ ] 产品描述（含关键词: laser cutting gas mixer, auxiliary gas system）
- [ ] 技术参数 PDF 上传
- [ ] 产品图片（白底高清）
- [ ] 联系信息 + gasmixtech.com 链接

### 费用
- 基础 listing: 免费（有限功能）
- Premium listing: 约 $2,000-5,000/年

## DirectIndustry (欧洲)

- 网址: directindustry.com
- 定位: 欧洲工业 B2B 展会和采购平台
- 多语言支持: 英/法/德/西/意 — 正好覆盖我们目标市场
- 特点: 虚拟展厅 + PDF 资料下载 + 询盘系统

### 注册准备
- [ ] 多语言产品描述
- [ ] 技术资料 PDF（每种语言一份）
- [ ] 产品视频链接（YouTube）
- [ ] 认证/资质文件

### 费用
- 基础: 免费注册
- 高级: 按行业和展示位置定价

## Made-in-China / Alibaba

- 适合被海外代理商/经销商发现
- 竞争激烈，需要持续运营
- 建议在 LinkedIn + ThomasNet/DirectIndustry 跑通后再考虑

## 运营节奏

| 阶段 | 行动 |
|------|------|
| 第1周 | ThomasNet + DirectIndustry 注册 |
| 第2周 | 上传完整产品信息 |
| 第3-4周 | 优化关键词和描述 |
| 持续 | 每月检查询盘，更新产品信息 |

## 与 LinkedIn 的协同

B2B 平台和 LinkedIn 形成**被动 + 主动**双引擎：
- **ThomasNet/DirectIndustry**: 买家主动搜索发现 → 被动获客。适合长尾询盘和品牌存在
- **LinkedIn**: 主动触达目标客户 → 主动获客。适合精准决策者触达和信任建设

**交叉引用**: 在 ThomasNet/DirectIndustry 公司介绍中提及 LinkedIn 页面；LinkedIn 帖子和公司页面中包含平台 listing 链接。两者互为信任背书。
